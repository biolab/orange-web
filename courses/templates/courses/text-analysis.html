Text analysis for social scientists

Social sciences no longer entail only 'manual' analysis of text. New, simple tools enable the analysis of large data and long texts in a very short time.

Apply

Why should I attend?

  • We will learn about key machine learning and data visualization technologies for digital humanities.
  • We will focus on content, not methods (no equations or programming).
  • Everything will be hands-on and practical.
  • We will learn how to apply new skills to teaching and research.
  • We will be using free tools you can take home after the workshop.
  • The course is given by the best lecturers in data mining. See our youtube channel for proof.

No coding knowledge required – this is a beginner workshop!

What will you learn?

  • Corpus-retrieval techniques
  • Text preprocessing and automatic syntactic annotation
  • Machine learning and text mining techniques for social sciences
  • Text clustering and segmentation
  • Topic modelling
  • Geotagging and sentiment analysis
  • Independent text analysis
